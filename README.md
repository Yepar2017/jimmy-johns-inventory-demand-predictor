# jimmy-johns-inventory-demand-predictor

## Overview
This project predicts weekly bread demand using machine learning to optimize inventory management and reduce waste.

## Business Problem
Over-ordering leads to waste, while under-ordering leads to lost sales. This model helps predict optimal inventory levels.

## Objectives
- Predict weekly bread usage
- Optimize inventory ordering
- Reduce waste and improve profitability

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
